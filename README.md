# 🚀 AWS EC2 Static Website Hosting

## 📌 Project Overview

This project demonstrates hosting a modern static website on AWS EC2 using Apache Web Server.

The website was deployed on an Ubuntu EC2 instance and configured using Linux commands.

---

## ☁️ Technologies Used

- AWS EC2
- Ubuntu Linux
- Apache2
- HTML/CSS

---

## ⚙️ Steps Performed

### 1. Launch EC2 Instance
Created Ubuntu EC2 instance on AWS.

### 2. Connect using SSH
```bash
ssh -i key.pem ubuntu@public-ip
```

### 3. Install Apache Web Server
```bash
sudo apt update
sudo apt install apache2 -y
```

### 4. Configure Website
Edited the default index.html file and deployed a custom landing page.

### 5. Access Website
Opened the website using the EC2 public IP.
